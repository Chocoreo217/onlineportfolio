Memory Game Android App

View Github
More info
GOAL:

A simple memory game with cute kitties. This is my first Android App in Java with Android Studio.

LEARNING POINTS:

  • Java and XML are the two main programming languages used in Android App development. Knowledge and mastery over these programming languages are prerequisites to developing an Android app. Proper understanding of Java and XML helped me a lot to build/develop this app.
  • It was very important to familiarize myself with the build automation tools as well as the integrated development environment before I start developing my app. I used Android app studio IDE and it helped me learn the basics and many other things that helped improve my code.
  • Application components are the essential building blocks of Android app development. Each of the components is a different point by which the system can enter the app. Although each one of them exists as its own entity and plays a specific role, there are some which depend on each other, and not all of them are actual entry points. There are different types of app components each serving a distinct purpose with a distinct life cycle which defines how it is created and destroyed.
